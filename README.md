## ğŸš€ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic understanding of HTML/CSS/JavaScript (for modifications)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR-USERNAME/MELODI.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd MELODI
   ```

3. **Open in browser**
   - **Option 1:** Double-click `index.html` to open directly
   - **Option 2:** Use a local server (recommended):
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js http-server
     npx http-server
     
     # Using PHP
     php -S localhost:8000
     ```
   - Open `http://localhost:8000` in your browser

### Usage

1. **Browse Music:** Explore featured tracks and playlists on the homepage
2. **Search:** Use the search bar to find specific songs or artists
3. **Play Music:** Click on any track to start playback
4. **Control Playback:** Use play/pause, skip, and volume controls
5. **Create Playlists:** Organize your favorite songs into custom playlists
6. **Save Favorites:** Mark tracks as favorites for quick access
